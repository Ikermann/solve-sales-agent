# SOLVE Sales Agent

An interactive sales training application that helps you practice and master the SOLVE framework for handling sales objections and guiding conversations to successful outcomes.

## Features

- **Interactive Role-Play**: Practice sales conversations with AI-powered prospects
- **SOLVE Framework Tracking**: Real-time progress tracking through the SOLVE methodology
- **Customizable Scenarios**: Choose from different prospect personas and industry challenges
- **Modern UI**: Clean, responsive interface with Canva-inspired design system

## Getting Started

### Prerequisites

- Node.js 16+ and npm installed on your system
- A modern web browser

### Installation

1. Clone the repository:
```bash
git clone https://github.com/[your-username]/solve-sales-agent.git
cd solve-sales-agent
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file in the project root and add your API keys (optional - needed for AI features):
```env
VITE_CLAUDE_API_KEY=your_claude_api_key_here
```

4. Start the development server:
```bash
npm run dev
```

5. Open your browser and navigate to `http://localhost:5173`

## Usage

1. Select a prospect persona and industry scenario
2. Review the SOLVE framework guide if needed
3. Start the practice session
4. Engage in conversation using the SOLVE methodology:
   - **S**pot the Pain
   - **O**utline Outcome
   - **L**imit Risk
   - **V**alue Pack
   - **E**xecute CTA
